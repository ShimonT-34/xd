Listas

Las listas pueden ser ordenadas (ol)(que significa "ordered list") o sin orden (ul)(que significa "un-ordered list"), y sus elementos se colocan sangrados con el "li" (que es "list item")

Tablas

td significa table data, si pones varios td se agruparan en una fila

tr significa table row, o fila de tabla

th significa table header y sirve para reconocer los encabezados de una tabla

Formularios

Los formularios empiezan con el < form > y pueden incluir varios elementos de entrada como < input >, < textarea >, < button >, etc.

Sin el atributo name no se envian los datos del input.

Es fundamental que los inputs esten asociados a un label (etiqueta)

Divisiones y Spans

Etiquetas como < span > permiten seleccionar una parte del párrafo, pero sin alterar el contenido del parrafo
